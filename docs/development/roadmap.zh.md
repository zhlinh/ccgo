## 优先级定义

- **P0（关键）**：阻塞核心功能或发布
- **P1（高）**：重要功能或重大改进
- **P2（中）**：增强功能，有价值但不紧急
- **P3（低）**：长期规划，锦上添花

## 如何贡献

我们欢迎贡献！以下是您可以帮助的方式：

1. **选择功能**：从 P1 或 P2 优先级中选择一项
2. **讨论**：开启 GitHub Discussion 或 Issue 讨论您的方法
3. **实现**：遵循我们的[贡献指南](contributing.md)
4. **测试**：确保您的更改在各平台上工作
5. **文档**：更新新功能的文档
6. **提交**：创建 pull request

详细指南请参阅[贡献指南](contributing.md)。
